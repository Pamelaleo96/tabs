¿Que son atributos data? Son atributos personalizados que nos permiten añadir datos a nuestros elementos HTML, que no necesariamente son visibles pero que nos ayudan mucho al momento de dar información adicional de dichos elementos.

¿Qué es un evento? Es cuando el usuario realiza una acción permitiéndole interactuar con el programa. A cada evento se le asocia un manejador (conocido como eventHandlers o eventListener) el cual se ejecutará cuando ocurra el evento.

Modificando contenido Los atributos innerHTML y textContent funcionan como métodos para obtener y establecer el valor de una propiedad específica.
